# GitHub Upload Steps

1. Download the GitHub-ready ZIP.
2. Unzip it locally.
3. Create a new GitHub repository.
4. Upload the unzipped contents, not the ZIP file itself.
5. Commit to `main`.
6. Go to **Settings → Pages**.
7. Select **Deploy from a branch**.
8. Branch: `main`; folder: `/root`.
9. Save and wait for Pages to deploy.

The game entry point is `index.html`.
