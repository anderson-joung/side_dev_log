# SJ Applied Tech Survival Tycoon v3 Design Notes

## 1. 새 UI 와이어프레임

밝은 제조 사무실 톤을 기본으로 한다. 배경은 흰색/연회색, 패널은 반투명 흰색, 위험 신호만 빨강/주황으로 처리한다.

```text
┌─────────────────────────────────────────────────────────────────────────────┐
│ TOP BAR                                                                     │
│ [회사명]        [날짜/생존일/전략]          [일시정지] [1x] [4x] [8x] [저장] │
│                                                                             │
│ [현금] [AR] [AP] [부채] [월 번] [압박]                                      │
├───────────────┬───────────────────────────────────────────────┬─────────────┤
│ LEFT RAIL     │ CENTER FULL ROOM VIEW                         │ RIGHT RAIL  │
│               │                                               │             │
│ 현장 방문      │ 현재 방문 중인 공간을 전체 화면으로 표시          │ 공간 상태    │
│ [현장]         │ - MCT/CNC 현장                                 │ - 핵심 지표  │
│ [조립실]       │ - 아쎄이 조립실                                 │ - 공간별 설명│
│ [사무실]       │ - 사무실                                       │             │
│ [설계실]       │ - 설계실                                       │ 공간 투자    │
│               │                                               │ - 현재 공간  │
│ 생존 액션      │ 이벤트 배너 / 토스트는 중앙 위·좌하단에만 표시      │   관련 부서  │
│ [클레임 무마]  │                                               │   업그레이드 │
│ [납기 압박]    │                                               │             │
│ [결제 홀드]    │                                               │             │
│ [채권 할인]    │                                               │             │
│ [대책회의]     │                                               │             │
│ [휴게실]       │                                               │             │
└───────────────┴───────────────────────────────────────────────┴─────────────┘
```

핵심 변경점:
- 한 화면에 모든 공간을 넣지 않는다.
- 좌측 상단은 공간 방문, 좌측 하단은 생존 액션으로 분리한다.
- 중앙은 현재 공간 하나만 크게 보여준다.
- 우측은 현재 공간과 관련 있는 상태/투자만 보여준다.
- 아래 HUD는 제거하고 주요 자원은 모두 상단으로 올린다.

## 2. 공간 4개별 게임 기능표

| 공간 | 감정 | 핵심 표시 | 주요 기능 | 연결 투자 |
|---|---|---|---|---|
| MCT/CNC 현장 | 긴장, 가동률, 사고 위험 | MCT 가동, CNC 가동, 정비 여력, 작업 대기 | 수주 물량의 실제 생산 기반, 산재/정비/외주 리스크 발생 | MCT·CNC 가공셀, 공구·소재·외주 |

| 사무실 | 정리된 공포, 책임 | 현금, AR/AP, 부채, 법무 리스크, 회수율 | 채권 할인, 결제 홀드, 금융/영업 판단 | 자금·회수, 영업·견적·CS, 대기업 벤더락 |

## 3. 조립실 36대 배치 시스템

SMT 스크린 프린터 36대 생산을 세 단계로 나눈다.

```text
Batch 1: 프레임 12대
  - 베이스 프레임, 외장, 기구 기준
  - 여기서 지연되면 전체 배치가 밀린다.

Batch 2: 내부 조립 및 PLC 12대
  - 배선, 센서, PLC, 내부 조립
  - 사기와 작업 숙련도의 영향을 크게 받는다.

Batch 3: Idle Run 12대
  - 무부하 운전, 반복 테스트, 출하 판정
  - 완료되면 36대 출하 이벤트 발생
```

### 수식 방향

```text
일일 조립 진행량 = 기본 진행량
              + MCT/CNC 가공셀 효과
              + SMT 조립·풀아쎄이 효과
              + 설계 표준화 효과
              + ERP/MES 효과
              × 사기/안전/압박 보정
```

### 불량/재작업

```text
불량률 = 기본 불량률
      + 압박 증가분
      + 안전 저하분
      - 설계 표준화 효과
      - ERP/MES 효과
```

재작업이 발생하면 해당 일의 생산 1칸은 진행되지 않고, 압박이 조금 오른다.

### 출하 보상

36칸이 모두 완료되면:
- 출하 횟수 +1
- 매출채권 증가
- 평판 소폭 증가
- 압박 소폭 감소
- 다음 36대 배치 시작

## 4. index.html 수정 지시안

실제 v3 index.html에 반영된 항목:

1. 화면 구조 변경
   - 기존 단일 회사 지도형 중앙 화면을 방문형 공간 구조로 변경
   - `currentRoom` 상태 추가
   - 좌측에 `현장 / 조립실 / 사무실 / 설계실` 버튼 추가

2. 밝은 UI 테마
   - 흰색/연회색 기반 패널
   - 위험/압박만 빨강·주황으로 강조
   - 전체 배경과 패널을 더 읽기 쉽게 변경

3. 상단 HUD 통합
   - 현금, AR, AP, 부채, 월 번, 압박을 모두 상단으로 이동
   - 하단 리소스 바 제거

4. 액션 버튼 축약
   - 클레임 무마
   - 납기 압박
   - 결제 홀드
   - 채권 할인
   - 대책회의
   - 휴게실

5. 공간별 우측 투자 필터링
   - 현장: MCT/CNC, 공구/소재/외주
   - 조립실: SMT 조립, ERP/MES
   - 사무실: 자금/회수, 영업/CS, 벤더락
   - 설계실: 설계 표준화, ERP/MES

6. 시간 속도 상승
   - 1일 진행이 이전보다 빠르게 체감되도록 `MS_PER_HOUR`를 단축
   - 1x/4x/8x는 유지

7. 업로드 이미지 적용
   - MCT/CNC 현장 배경 적용
   - 한국 산업단지 배경 적용
   - 액션 아이콘 시트 crop 적용
   - 공간 방문 아이콘 crop 적용

## 5. 플레이어가 즐거움을 느끼는 지점

이 게임의 재미는 단순한 joy보다 "통제 가능한 지옥"에 가깝다.

- 완전히 고칠 수는 없지만 오늘 하루를 버틸 수 있다.
- 좋은 선택이 아니라 덜 나쁜 선택을 고른다.
- 조립실 36칸이 차오르는 모습은 작은 성취를 준다.
- 사무실은 숫자의 공포를 준다.
- 설계실은 지금 손해를 감수하고 미래를 덜 망치는 감각을 준다.
- 귀여운 동물 캐릭터와 잔혹한 회사 생존 압박 사이의 간극이 블랙코미디가 된다.
