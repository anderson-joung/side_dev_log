# 이번 달까지만 버텨주세요 - AI Handoff

English title: Please Let Us Survive This Month

## Current direction
- Korean small manufacturing survival management simulation.
- Generic SME manufacturing, not a specific product/company-branded game.
- Core loop: production flow, cash flow, delivery pressure, design fixes, monthly reports.
- Goal: survive the current month.

## v4.4.12 notes
- Removed old project/company branding from visible game text.
- Production flow: order -> frame -> assembly -> PLC/electrical -> QC -> idle/shipment.
- Overtime triggers when order or WIP exceeds 12.
- Monthly report now includes simplified balance sheet, income statement, and manufacturing cost statement.
